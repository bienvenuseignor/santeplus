# Santé+ Bénin

Application fullstack de gestion de santé, rendez-vous, wallet et paiements pour le projet Santé+.

## Démarrage local

Prérequis : Node.js

1. Installer les dépendances :
   `npm install`
2. Lancer l’application :
   `npm run dev`
3. Ouvrir l’interface sur :
   `http://localhost:3000`
